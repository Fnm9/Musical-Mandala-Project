## Descirption 
The musical mandala project will create a colorful mandala that changes colors based on the movement of the cursor and plays meditation music. This project is relevant to digital arts and media because it works with motion graphics and visual design. 

## Features
- Feature 1
	- This project will have an symmetry system. In this project for every cursor movement it will calculate distance from center and duplicate that point in a circular pattern. 
- Feature 2
	- Also the project will have audio playback system. Will use pygame.mixer to play music 
- Feature N 
	- In addition the project will have a drawing system. Will draw each rotated point every frame.
